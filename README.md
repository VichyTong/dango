## üòÆ Highlights
Dango is a mixed-initiative system that enables users to generate
data wrangling scripts. Compared to existing tools, Dango enhances user communication of intents by 
(1) üí° enabling users to clarify their intents by answering LLM-posed clarification questions,
(2) üìù allowing users to demonstrate on multiple tables, and
(3) üîÑ providing multiple feedback modalities to help users evaluate and refine the data wrangling scripts generated by LLM.

In a within-subjects, think-aloud study (n=25), the results show that Dango‚Äôs features can significantly improve intent specification, accuracy, and efficiency in data wrangling tasks.

## üõ†Ô∏è Requirements and Installation

### Frontend

* [Node.js](https://nodejs.org/en/download/) >= v20.17.0

```bash
cd dangle-vite
npm install
```

### Backend

* [Python](https://www.python.org/downloads/) >= v3.10

We suggest using [Conda](https://docs.conda.io/en/latest/miniconda.html) to manage the Python environment.

Create conda environment:
```bash
conda create -n dango python=3.10 -y
conda activate dango
cd dango-backend
pip install -r requirements.txt
```

### Set OpenAI API
```bash
export OPENAI_API_KEY=<YOUR_API_KEY>
```
